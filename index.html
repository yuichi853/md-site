Markdown Toolkit

Markdownを使って、簡単Webサイト制作！

MD Site は、Markdownの使い方からWebサイト制作までを学べる実践型サイトです。 Markdownのシンプルな記法や利点を紹介しながら、JavaScriptによるHTML変換機能を備えたテンプレートを配布しています。 文章づくりに集中したまま、自分のサイトを簡単に立ち上げることができます。

Markdownの基本を知る 制作ステップを見る

workflow

  • テンプレートをダウンロード

features

ビルドツール不要で、HTMLとCSSの基礎だけで動作。GitHub Pagesへの無料公開にも対応しています。

01

Markdownの魅力を知る

軽量で扱いやすく、どんな環境でも動作するMarkdownの魅力を、実例を交えてわかりやすく紹介します。

02

初心者向けの解説

基本記法から記事の公開まで、ステップごとに解説。はじめてでも安心して学べます。

03

テンプレートで簡単公開

MarkdownをHTMLに変換する機能を組み込んであるテンプレートをそのまま使ってサイトを公開できます。

テンプレートではMarkdownからHTMLへの変換機能に加え、デザインのサンプルも複数用意しています。 まずはテンプレートをダウンロードして触ってみてください！
